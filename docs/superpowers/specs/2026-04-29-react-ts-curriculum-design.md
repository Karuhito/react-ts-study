# React + TypeScript 学習カリキュラム設計

## 学習者プロフィール

- **JS理解度:** 概念（DOM・async/await）は理解しているが、コードで記述した経験は少ない
- **最終目標:** 路面状況を報告・共有できるWebアプリの開発
- **学習時間:** 1日3時間
- **学習スタイル:** 小さな課題を積み重ねながら手を動かす（アウトプット重視）
- **英語:** 読めるが日本語の方が理解しやすい

---

## アプローチ

**JS実践 → TypeScript → React** の順で進める。

「概念はわかるがコードが書けない」状態でTypeScriptに入ると、型エラーとJS構文エラーが同時に発生して混乱しやすいため、まずJSを「書ける」状態にしてからTSに進む。

---

## Phase 0: JS実践（約1週間 / 21時間）

**目標:** 「概念はわかる → コードで書ける」に変える

| 日程 | 学習内容 | 課題 |
|------|---------|------|
| 1日目 | 変数・関数・アロー関数・配列メソッド（map/filter/find） | 路面状況データを配列で操作する |
| 2日目 | オブジェクト・分割代入・スプレッド構文 | 報告データのオブジェクトを作る |
| 3日目 | DOM操作（querySelector・innerHTML） | 報告一覧をHTML上に表示する |
| 4日目 | イベントリスナー（click・submit） | 投稿フォームの送信処理 |
| 5日目 | fetch・async/await・Promiseを書く | 公開APIからデータを取得して表示 |
| 6〜7日目 | 総まとめ課題 | 路面状況を入力→一覧表示するミニアプリ（Vanilla JS） |

---

## Phase 1: TypeScript基礎（約2週間 / 42時間）

**目標:** 型を使ってコードを安全に書けるようになる

| 日程 | 学習内容 | 課題 |
|------|---------|------|
| 1日目 | TSのセットアップ・基本型（string/number/boolean/null） | Phase 0の変数にすべて型を付ける |
| 2日目 | 配列の型・タプル・型推論 | 路面状況データの配列に型を付ける |
| 3日目 | オブジェクト型・`interface`・`type` | `RoadReport`型を定義する |
| 4日目 | Union型・Optional（`?`）・リテラル型 | 状況の種類（`"良好" \| "悪化" \| "危険"`）を型で表現 |
| 5〜6日目 | 関数の型付け・戻り値の型 | 報告を追加・削除する関数に型を付ける |
| 7日目 | 型アサーション・`as`・`unknown`・`any`を避ける理由 | 外部APIレスポンスに型を付ける |
| 8日目 | ジェネリクス入門 | `ApiResponse<T>` 型を作る |
| 9〜10日目 | `enum`・`readonly`・utility types（`Partial` / `Pick`） | 報告フォームの入力データ型を整理 |
| 11〜14日目 | 総まとめ課題 | Phase 0のVanilla JSアプリをTSに書き直す |

**注意:** 毎日 `tsconfig.json` の `strict: true` で動かす。

---

## Phase 2: React + TypeScript基礎（約3週間 / 63時間）

**目標:** Reactのコンポーネント設計とデータの流れを理解する

| 日程 | 学習内容 | 課題 |
|------|---------|------|
| 1〜2日目 | Vite + React + TSセットアップ・JSX/TSX | Hello World → コンポーネント分割 |
| 3〜4日目 | Props（型付き）・コンポーネントの分割 | `ReportCard`コンポーネントを作る |
| 5〜7日目 | `useState`・フォーム制御 | 路面状況の投稿フォームを作る |
| 8〜9日目 | リスト表示・`key`・条件付きレンダリング | 報告一覧を表示・フィルタリング |
| 10〜12日目 | `useEffect`・データフェッチ（fetch + async/await） | APIから報告を取得して表示 |
| 13〜15日目 | イベント型・フォームの型付け | フォームに完全な型を付ける |
| 16〜21日目 | 総まとめ課題 | 投稿・一覧・削除ができるReactアプリ |

## Phase 3: React中級（約3週間 / 63時間）

**目標:** 実際のアプリに必要な技術を身につける

| 日程 | 学習内容 | 課題 |
|------|---------|------|
| 1〜4日目 | React Router v6（ページ遷移・URLパラメータ） | 一覧ページ・詳細ページを作る |
| 5〜7日目 | Context API（グローバル状態） | ログイン状態を管理する |
| 8〜10日目 | カスタムフック | `useRoadReports()`フックを作る |
| 11〜14日目 | TanStack Query（非同期データ管理） | APIデータのキャッシュ・再取得 |
| 15〜21日目 | Leaflet.js（地図・ピン表示） | 地図上に路面状況ピンを立てる |
